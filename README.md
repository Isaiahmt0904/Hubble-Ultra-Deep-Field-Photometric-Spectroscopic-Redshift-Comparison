# Hubble-Ultra-Deep-Field-Photometric-Spectroscopic-Redshift-Comparison

Hubble Ultra Deep Field analysis comparing photometric and spectroscopic galaxy redshifts using HST imaging and catalog cross-matching.

\## Data



This project uses HST imaging and catalog data.



Due to file size constraints, raw FITS files are not included in this repository.



To reproduce results:

\- Download data from MAST (Hubble Legacy Archive)

\- Place files in:

&#x20; - `Downloaded Data/`

&#x20; - `mastDownload/`
